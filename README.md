# ☁️ SkyGlass - App de Clima ☀️

Este proyecto es una aplicación de clima desarrollada con React, diseñada para practicar el consumo de APIs y la creación de interfaces modernas y responsivas.

## 🛠️ Tecnologías principales

- **React**: Biblioteca principal para la construcción de la interfaz de usuario.
- **Tailwind CSS**: Utilizado para los estilos y la maquetación responsiva y moderna.
- **OpenWeather API**: Fuente de los datos meteorológicos en tiempo real.
- **react-hook-form**: Para la gestión y validación eficiente de formularios.
- **react-icons**: Para el uso de iconos modernos y personalizables.

## 🚀 Instalación

1. Clona el repositorio:
	```bash
	git clone https://github.com/emiarias/skyGlass
	cd skyGlass
	```
2. Instala las dependencias:
	```bash
	npm install
	```
3. Crea una API key gratuita en [openweathermap.org/api](https://openweathermap.org/api) y configúrala en un archivo `.env` si es necesario:
	```env
	VITE_WEATHER_API_KEY=tu_api_key
	```
4. Inicia la aplicación:
	```bash
	npm run dev
	```

## ✨ Estructura principal

- `src/components/WeatherCard.jsx`: Componente principal para mostrar el clima actual.
- `src/components/WeatherStats.jsx`: Componente para mostrar estadísticas adicionales.
- `src/App.jsx`: Estructura principal de la app.
